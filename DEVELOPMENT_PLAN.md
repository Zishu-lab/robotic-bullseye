# 机器人靶心识别与内容分析系统开发计划

## 0. 项目背景
- **目标赛事**：瑞抗机器人开发者大赛 / 中国机器人及人工智能大赛
- **核心挑战**：
  1. 远距离（小目标）识别率低
  2. 环境干扰大
  3. 识别距离受限
- **主要目标**：
  1. 检测靶标位置（克服远距离和干扰）
  2. 识别靶心内容（CIFAR-100 物品类别）

## 1. 技术路径

```
输入图像（远距离、有干扰）
    ↓
┌───────────────────────────────────────┐
│  预处理增强                            │
│  - 黑白圆环特征提取（增强靶标特征）     │
│  - 对比度增强（克服远距离模糊）         │
│  - 形态学处理（去除环境干扰）           │
└───────────────────────────────────────┘
    ↓
┌───────────────────────────────────────┐
│  小目标检测（YOLO 优化）               │
│  - P2 小目标检测层                     │
│  - 高分辨率输入                        │
│  - 针对远距离训练                      │
└───────────────────────────────────────┘
    ↓
┌───────────────────────────────────────┐
│  中心区域裁剪                          │
│  - 基于检测框定位中心                  │
│  - 尺寸标准化                          │
└───────────────────────────────────────┘
    ↓
┌───────────────────────────────────────┐
│  CIFAR-100 分类                        │
│  - 识别中心物品类别（100 类）          │
└───────────────────────────────────────┘
    ↓
输出：靶标位置 + 物品类别
```
